Texte

Tagtäglich erstellen wir Texte, bearbeiten sie, wollen sie austauschen. Nicht immer funktioniert das verlustfrei.

TEI

Format
  • Texte erstellen,
  • be- und verarbeiten,
  • speichern und
  • austauschen.

Seit es Computer gibt, besteht ein Interesse, mit diesen Texte zu erstellen, zu speichern und diese dann auch weiter verarbeiten zu können.

Mit TEI erfassen wir Texte. Dabei ist egal in welcher Form oder welchem Medium sie vorliegen.

Das machen wir in strukturierter Form, so dass Computer mit diesen Dokumenten umgehen können.

TEI Anpassungen

Because the TEI Guidelines must cover such a broad domain and user community, it is essential that they be customizable: both to permit the creation of manageable subsets that serve particular purposes, and also to permit usage in areas that the TEI has not yet envisioned. Customization is a central aspect of TEI usage and the Guidelines are designed with customization in mind.
  • Lite
    • TEI Lite, the most widely used TEI customization; includes basic elements for simple documents
  • Tite
    • A constrained customization designed for use by keyboarding vendors.
  • Bare
    • TEI Absolutely Bare, a very barebones schema with the absolute minimum of elements
  • All (?)
  • …

Damit die Richtlinien auch anpassbar bleiben, wurde ein eigenes Format für die Erfassung des Schemas entwickelt: ODD (One Document does it all).

Prozessierungsmodell (PM)

TEI-Untermenge

Basierend auf einer empirischen Erhebung der genutzen Elemente in DTA und EEBO wurden ein Subset erstellt. Dabei wurde nur der Textteil (<text>) der TEI-Dokumente ausgewertet. So ist auch im TEI Simple-Schema der teiHeader nicht eingeschränkt und das Schema bezieht sich nur auf tei:text.

Sonderfall teiHeader

The default set of elements for the header are loaded using the header module. In addition, elements from other modules are loaded, if they are tagged in the classification as being needed for the header only.

Was passiert mit den beiden anderen "Text-Stellvertretern" <facsimile> und <sourceDoc>?

Sie werden initial geladen.
